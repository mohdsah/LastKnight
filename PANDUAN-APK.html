📱 Panduan Buat APK

Pahlawan Terakhir — Guna PWABuilder

Masa: ~15 minit | PERCUMA | Tiada coding

🔁 Gambaran Keseluruhan

Netlify (host game) → PWABuilder (buat APK) → Install di Android

BAHAGIAN A — Deploy ke Netlify

① Pergi ke Netlify
https://app.netlify.com
Log masuk atau daftar akaun percuma.
② Upload fail game
Klik "Add new site" → "Deploy manually"

Drag & drop fail ko-game-v7.zip ke kotak yang disediakan.
Tunggu 30-60 saat untuk deploy selesai.
③ Dapat URL
Selepas deploy, Netlify bagi URL seperti:
https://amazing-game-abc123.netlify.app
Anda boleh tukar nama dalam Settings → Site name:
https://pahlawan-terakhir.netlify.app
④ Test dalam browser
Buka URL dalam Chrome Android. Game mesti muat dengan betul.
✅ Jika game berjalan = sedia untuk APK

BAHAGIAN B — Buat APK dengan PWABuilder

① Pergi ke PWABuilder
https://www.pwabuilder.com
② Masuk URL game
Taip URL Netlify anda dalam kotak carian.
Contoh: https://pahlawan-terakhir.netlify.app

Klik "Start" dan tunggu analisis selesai (~10 saat).
③ Semak skor PWA
PWABuilder akan tunjuk skor untuk 3 kategori:

KategoriSkor Anda (dijangka)Minimum APK
Manifest~100/10070/100
Service Worker~100/10070/100
Security (HTTPS)100/100100/100
✅ Semua skor game ini sudah dioptimumkan
④ Generate Android Package
Klik "Package For Stores" → Pilih "Android" → Klik "Generate Package"

Isi maklumat APK:
FieldNilai
Package IDcom.mohdsah.pahlawanterakhir
App NamePahlawan Terakhir
Version Name6.0
Version Code6
Display Modefullscreen
Orientationany
SigningMine (PWABuilder sign untuk anda)
⑤ Download APK
Klik "Download Package".
Anda akan dapat fail pahlawan-terakhir.apk (atau .zip).

BAHAGIAN C — Install APK di Android

① Aktifkan Unknown Sources
Settings → Security → Unknown Sources: ON
(atau Settings → Apps → Special app access → Install unknown apps)

⚠️ Ini hanya perlu untuk install APK sendiri. Play Store APK tidak perlukan ini.
② Transfer APK ke telefon
Hantar fail .apk ke telefon melalui:
• WhatsApp / Telegram (hantar sebagai fail)
• Google Drive
• USB cable
• Email attachment
③ Install
Tap fail .apk dalam telefon → Install → Open

✅ Game akan muncul sebagai app dalam launcher Android!

BAHAGIAN D — Upload ke Google Play Store (Pilihan)

① Daftar Google Play Console
https://play.google.com/console
Bayaran pendaftaran sekali: USD $25
② Generate .AAB (bukan .APK)
Di PWABuilder, pilih "Google Play" (bukan Android).
Ini akan generate .aab (Android App Bundle) yang diperlukan Play Store.
③ Upload ke Play Console
Create new app → Upload .aab → Isi maklumat:

FieldContoh
App namePahlawan Terakhir
Short descriptionKnight Online Classic Mobile RPG Malaysia
CategoryGames > Role Playing
RatingTeen (13+)
